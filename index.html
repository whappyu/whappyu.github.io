忠于内心，逆风奔跑

发表于 2018-04-11   |   分类于 分享ing   |  

距离考研结果出来已过去8天，不知道为什么，我觉得这8天如此的漫长。经过一年的学习，我如愿的考上了自己想要去的学校。作为一个还有两个月就要毕业的老学长，应老师的建议，有一些想法对大一大二的童鞋们说说，希望对你们有些帮助。
本来这篇文章的题目写的是做一个有故事可讲的大学生，我总觉得大学不能太平淡，要不然以后回想起来没有一两件可以吹的故事，那就太遗憾了。但在大一大二的时候，有时候我真的不知道除了上课以外，自己要做什么，又能去做些什么，我觉得大一大二的很多同学也会和我有类似的想法。我来自河南，当年高考成绩出来的时候，彻底崩溃，那种感觉我永远也忘不掉。那时的很多天里，我完全的失去了方向。高中时曾经带过我的班主任给我打了多次电话，建议复读一年，后来由于各种原因吧，我选择了进入大学。
来到大学的第一天，报道的时候每个人可以在一个新生签名墙上写一句话，我现在非常清楚的记得我写的是“希望这里是我梦想起飞的地方”，然后把我的名字大大的写在旁边，然后就开始了大学生活。我来大学前目标就是考研，虽然那时候自己对考研一点概念都没有，但还是想让时间尽快的到考研的时候。但很多事情是急不来的，后来我这种想法也被逐渐丰富的大学生活给冲淡了。所以学长真的真的很建议多丰富自己的大学生活，出去走走看看，我现在越来越认同“读万卷书，行万里路”，每一次出发都会有不同的收获，写到这里，突然想起来复试的时候，我的英语自我介绍有一句话“I like traveling because I can learn a lot of diverse culture”.开学首先就是军训，那时候还是挺期待的，我比较喜欢唱歌，在训练间隙也会被教练叫出去唱那么一小段，现在想想，当时唱的是啥啊，真的是献丑啊哈哈。不过这个经历也算是成了我大学锻炼胆量的第一步吧，建议你们多和人交流，勇于和人面对面交流，不要总是网上虚拟世界的交流。后来我加入了校外联部，对我的影响还挺大的，因为我们每次和别人谈赞助，自己就多一次锻炼的机会，很感谢当时带我们的学长学姐还有一起奋斗的小伙伴们。

简短的杭州行

发表于 2016-11-21   |   分类于 在路上   |  

11月17日

   中午吃过饭就匆忙和其他成员集合，目的地杭州，参加2016全国大学生移动应用创新大赛。经过一下午的车程，晚上七点左右到达目的地，在住宿的地方简单的收拾之后，开始准备吃晚饭。第一次品尝浙江菜， 确实不错，给人一种很细腻的感觉。晚上8点，去比赛场地彩排，有点开始紧张了，第一次参加这么大的比赛，但也想取得一点成绩，晚上回去后又反复的练习,到了凌晨一点终于可以睡觉了。

11月18日

  定的闹钟是六点的，但在五点的时候就醒了，又起来准备，七点半吃完早饭就去比赛地点，到比赛开始之前的 这段时间是最难熬的，参赛者们都在现场来回的走动以缓解紧张。九点比赛开始，在一些致辞之后，第一个参赛代表上场，终于到第七个了，在下面的时候还是挺紧张的，上去之后好了很多，路演完毕终于松了一口气，完成答辩后我们的比赛也就正式结束。

结语：

  虽然没进8强，但还是收获了很多，在比赛的时候，坐我旁边的一个学长说这种场面

Next主题，使用hexo+d部署github pages,出现空白页的解决方法

发表于 2016-11-13   |   分类于 分享ing   |  

自己差点换主题，还好找到了解决方法

  前两天我看到了一个非常实用的app,而且界面简洁，风格也很喜欢，也想给大家推荐一些实用的神奇软件，或者一些插件。可是当我准备写的时候，next主题出问题了（前期我并不知道是主题的问题，浪费了一个晚上在搞配置方面，不过也有收获，就是现在能闭着眼使用github+hexo了），晚上睡觉前无意去next主题代码托管的地方看了一下，这才知道大家遇到了同样的问题，接下来就给大家说一下解决的方法吧

首先作者说是手动将source/vendors目录修改为source/lib，同时修改主题配置文件_config.yml，将里面的_internal:vendors改为_internal:lib，可是试了一下还不行

然后看到github的官方回复邮件，我就在主目录第一个文件.deploy_git加上一个名为.nojekyll的空文件就好了，格式为.txt就行。

从此给自己一个提醒，有问题最好去Next主题官网看一下评论
